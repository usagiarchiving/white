대화 내용은 어떻게 넣나요?
대화를 하나씩 칠 필요 없이, 대본을 복사해서 붙여넣으면 자동으로 말풍선이 생성됩니다.
에디터가 인식할 수 있는 대본 포맷은 다음과 같습니다.
[기본 대화 인식 포맷]
- [이름: 내용] (예: [A: 안녕?])
- [이름: 내용 (끝에 닫는 대괄호 생략 가능. 예: [A: 안녕?)
- [이름] 내용 (예: [A] 안녕?)
[상대 메시지 대화 인식 포맷]
- [내용] (예: [밥 먹었어?])
- ✉️ 내용 (예: ✉️ 밥 먹었어?)
[날짜 및 시간 인식 포맷]
- 날짜 단독: [2026년 01월 15일], [2026.01.15], [01-15], [01월 15일] 등 대괄호 안의 날짜 형식
- 시간 단독: [오전 11:20], (PM 11:20), [11:20] 등 대괄호나 소괄호 안의 시간 형식
- 일기/로그형 시간+내용 동시 인식: [오전 11:20] 내용 (예: [오전 11:20] 일어났다.) -> 이 경우, 중앙에 '오전 11:20' 시간 라인이 찍히고 그 아래에 '상대방 메시지'로 "일어났다."가 찍힙니다.
수동으로 하나하나 수정하기 보다는 AI에게 아래와 같은 프롬프트와 함께 대화 내용을 입력하는 것을 추천드립니다.
▶ 프롬프트
아래 대화 내용을 [이름: 대사] 형식으로 일괄 변환해 줘.
[필수 규칙]
1. 원본 대사의 내용, 말투, 기호, 띄어쓰기는 절대 1글자도 수정하거나 요약하지 말고 100% 원본 그대로 유지할 것.
2. 다른 설명 없이 변환된 텍스트만 복사가 간편하게 코드 블록으로 출력해 줘.
(2명의 대화일 경우) 3. 대화의 뉘앙스를 파악하여 캐릭터를 구분한다. 첫 대사가 (이 부분을 수정해서 실제 대사를 넣으세요)인 캐릭터 이름을 A, 상대 캐릭터의 이름은 B로 설정.
각 블록들은 어떻게 사용하나요?
매번 새로 입력할 필요 없도록, 자주 쓰는 설정은 브라우저 로컬 스토리지에 자동으로 저장됩니다. 새로고침을 해도 설정이 유지됩니다.
- '나' / '상대' 트리거: 대본을 넣을 때 시스템이 "누가 한 말인지" 구별하는 기준 이름입니다. (기본값: 나 = A / 상대 = B)
- 상대방 이름 & 프로필 URL: 채팅방 맨 위와 상대방 말풍선 옆에 뜰 이름과 사진입니다. (기본값 이미지: 토끼 사진)
- 👆 프로필 헤더 / 👇 채팅 입력창: 클릭하면 실제 아이폰처럼 상단 프로필과 하단 텍스트 입력창 디자인이 추가됩니다.
- 🎨 말풍선 색상: '나'의 말풍선 색상을 고릅니다. 색상을 고르고 [저장]을 누르면 동그란 팔레트에 색이 수집되며, 언제든 클릭해서 바꿀 수 있습니다. (목록에서 지우고 싶을 땐 우클릭 하세요!)
- 💬 딜레이: V1(애니메이션 켜짐) 모드일 때, 말풍선이 톡, 톡, 하고 나타나는 시간 간격을 0.7초부터 2초까지 디테일하게 설정할 수 있습니다.
- 📅 날짜/시간 & 👀 읽음 표시: 수동으로 특정 시간이나 '읽음' 마크를 넣고 싶을 때 사용합니다. 날짜 칸을 비우고 시간만 넣으면 시간 라인만 깔끔하게 추가됩니다.
- 🖼️ 이미지 추가: 대화 중간에 사진을 보내고 싶다면 이미지 URL을 넣고 [상대 사진] 또는 [내 사진]을 누르세요.
추가한 메시지는 어떻게 수정하나요?
- 나 / 상대 변경: [나], [상대] 버튼 한 번으로 말풍선의 방향과 색상을 즉시 뒤집을 수 있습니다.
- 편집 버튼: 대사 내용이나 딜레이 시간을 다시 수정하고 싶을 때 누릅니다. 텍스트 창이 열리면 내용을 고치고 [완료]를 누르세요.
- ⬆️ 위로 추가 / ⬇️ 아래로 추가: 대화 중간에 무언가 빼먹었다면 이 버튼을 누르세요. 깔끔한 모달 팝업창이 뜨며 중간에 텍스트, 이미지, 날짜, 읽음 표시 등을 쏙 끼워 넣을 수 있습니다.
- 삭제 버튼: 불필요한 헤더나 메시지를 즉시 지울 수 있습니다.
V1과 V2의 차이는 무엇인가요?
- V1 (애니메이션 켜기): 말풍선이 아래에서 위로 올라가는 애니메이션 추가. 개인 홈페이지에 통째로 백업하거나, 화면 녹화를 통해 영상으로 만들 때 추천.
- V2 (애니메이션 끄기): 애니메이션 없이 모든 대화가 한 번에 완성된 상태로 멈춰서 출력됩니다. 단순히 화면 캡처용으로 쓸 때 추천.
어떻게 백업하고 사용하나요?
완료하셨다면 캡쳐 후 사용, 또는 가장 아래쪽의 [코드 복사하기] 버튼을 누르면 HTML 코드가 복사됩니다.
- 티스토리 / 포스타입 백업 : 글쓰기 모드를 '기본'에서 'HTML 모드'로 바꾼 뒤 복사한 코드를 그대로 붙여넣으면 됩니다.
- 개인 홈페이지: 개인 웹호스팅(Naru pub 등)을 쓴다면 텍스트 에디터나 배포용 HTML 파일에 붙여 넣으면 됩니다.
- HTML 미리보기 & 캡처: html 미리보기 사이트를 이용하거나, 에디터의 3단 미리보기 화면 자체를 스크린샷 찍어서 이미지 형태로 소장해도 좋습니다.
