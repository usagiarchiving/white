(1) 일괄 변환 (텍스트 입력)

- 지문과 "대사"가 섞인 문단을 넣으면 자동으로 분리됩니다.
- *내용* 양쪽에 별표가 있으면 나레이션으로 자동 인식합니다.
- ``` 마크다운 코드블록을 넣으면 HTML 블록으로 자동 변환됩니다.
- 마크다운 이미지 ![이미지](링크)는 깔끔하게 이미지 블록으로 빠집니다.
- --- 또는 *** 를 입력하면 회색 실선 구분선이 추가됩니다.
- 문장 맨 앞에 이모티콘(🗓️, 💍 등)이 오면 [상태창]이 됩니다.
- 포스트잇: 내용 또는 폴라로이드: 링크|날짜|캡션 을 입력하면 감성 블록이 생성됩니다.
+ ~~취소선~~, ++밑줄++, ==형광펜== 등 특수 기호 변환도 지원합니다.
