(1) 일괄 변환 (텍스트 입력)

- 지문과 "대사"가 섞인 문단을 넣으면 자동으로 분리됩니다.
- **문장** 양쪽에 별표가 두 개 있으면 나레이션으로 자동 인식합니다. (단어는 자동 삭제)
- ``` 마크다운 코드블록을 넣으면 HTML 블록으로 자동 변환됩니다.
- 마크다운 이미지 ![이미지](링크)는 깔끔하게 이미지 블록으로 빠집니다.
- [턴 숫자] 및 ---나 - 기호는 자동으로 삭제됩니다.
- 문장 맨 앞에 이모티콘(🗓️, 💍 등)이 오면 [상태창]이 됩니다.

(2) 캐릭터 설정 & 수동 블록 추가

▶ 제3자 (Custom) 캐릭터 일괄 관리 (클릭하여 열기)

에디터에 추가된 제3자 캐릭터가 자동으로 표시됩니다. 이미지 링크나 색상을 변경하고 [일괄 적용]을 누르세요.

(3) 내용 편집 공간

텍스트를 쓰면 우측에 실시간 반영됩니다. 상태창은 미리보기 창에서 직접 편집하세요.
💡 팁: (3), (4) 창 어디서든 Enter를 치면 새 블록으로 분리되고, Shift+Enter를 치면 단순 줄바꿈이 됩니다.

(4) 미리보기 확인 및 직접 수정

박스 안의 글자를 직접 클릭해 수정할 수 있습니다. 마우스를 올리면 [+ 공백 줄 추가] 버튼이 보입니다.

(5) 최종 HTML 확인 및 복사

이전에 복사했던 최종 HTML을 아래 빈 칸에 붙여넣고 [동기화하기]를 누르면 다시 편집할 수 있습니다.
